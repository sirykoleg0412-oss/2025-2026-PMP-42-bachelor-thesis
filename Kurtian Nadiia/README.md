# Advanced MHT Tracker: Robust Drone Tracking

**Коротко про проєкт**
Це **програмний комплекс для автоматичного супроводу БПЛА**, створений для вирішення проблеми **втрати цілі під час тривалих оклюзій** (коли об'єкт зникає за перешкодами: деревами чи будівлями).

**Як це працює?**
В основі лежить **Метод множинних гіпотез (MHT)**:
* **Генерація сценаріїв:** Система одночасно прораховує **декілька потенційних траєкторій руху** об'єкта.
* **Механізм Last Stand Fallback:** «Запобіжник», що дозволяє супроводжувати ціль навіть за **повної відсутності візуального контакту до 90 кадрів**.
* **Гібридна верифікація:** Алгоритм аналізує як **структурні контури (детектор Кенні)**, так і **напівтонові характеристики**, забезпечуючи стійкість до змін освітлення.

**Як використовувати?**
1. **Запуск:** виконайте команду `python main.py --video your_video.mp4` у терміналі.
2. **Ініціалізація:** на першому кадрі **виділіть дрон мишкою**, щоб задати початкові координати.
3. **Керування:**
    * **Зелені рамки** — альтернативні сценарії руху.
    * **Помаранчева рамка** — найбільш вірогідна ціль.
    * Натисніть **'q'**, щоб вийти з програми.

**Технічна база**
* **Мова:** Python 3.13.
* **Бібліотеки:** OpenCV, NumPy.
* **Алгоритмічне ядро:** MHT, Template Matching, експоненційне згладжування.
