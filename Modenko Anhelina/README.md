# Код до бакалаврської роботи — Ангеліна Моденко

У цій директорії розміщено матеріали до бакалаврської роботи студентки групи ПМП-42 Ангеліни Моденко.

Проєкт **DelaunayForge** підключено до цього репозиторію як Git submodule. Це означає, що в репозиторії `student-applications` зберігається посилання на окремий репозиторій із кодом проєкту, а не повна копія всіх файлів.

## Репозиторій проєкту

Основний репозиторій проєкту:

https://github.com/anshelino0606/DelaunayForge

У межах репозиторію `student-applications` проєкт підключено за шляхом:

    2025-2026/PMP-42/bachelor-thesis/Anhelina Modenko/DelaunayForge

## Призначення проєкту

**DelaunayForge** — це інтерактивний C++ застосунок для побудови обчислювальних областей, генерації триангуляцій, задання граничних умов та проведення чисельних експериментів, пов’язаних із методом скінченних елементів і диференціальними рівняннями в частинних похідних.

У межах бакалаврської роботи проєкт використовується для моделювання та аналізу задач на перфорованих і фрактальних областях, зокрема на областях типу килима Серпінського.

## Основний проєкт для відкриття

Основні дані для відтворення експериментів розміщено в директорії:

    DelaunayForge/fem_project/sierpinski_4

Якщо у поточній версії репозиторію директорія має попереднє написання назви, слід відкрити:

    DelaunayForge/fem_project/sierpisnki_4

## Як склонувати репозиторій разом із submodule

Оскільки `DelaunayForge` підключено як Git submodule, для повного клонування репозиторію потрібно виконати:

    git clone --recursive https://github.com/kpm-lnu/student-applications.git

Якщо репозиторій уже було склоновано без submodule, потрібно перейти в корінь репозиторію `student-applications` і виконати:

    git submodule update --init --recursive

Після цього директорія `DelaunayForge` буде завантажена всередині шляху:

    2025-2026/PMP-42/bachelor-thesis/Anhelina Modenko/

## Як зібрати проєкт

Перейдіть у директорію проєкту:

    cd "2025-2026/PMP-42/bachelor-thesis/Anhelina Modenko/DelaunayForge"

Згенеруйте конфігурацію збірки за допомогою CMake preset:

    cmake --preset release-bgfx

Зберіть проєкт:

    cmake --build --preset release-bgfx

Після успішної збірки виконуваний файл має бути створений у директорії:

    out/build/release-bgfx/

Очікувана назва виконуваного файлу:

    DelaunayForge

## Як відтворити експеримент

Щоб відтворити експеримент, потрібно виконати такі кроки.

1. Склонуйте репозиторій разом із submodule або виконайте `git submodule update --init --recursive`, якщо репозиторій уже було склоновано.
2. Зберіть проєкт за допомогою CMake preset `release-bgfx`.
3. Запустіть виконуваний файл `DelaunayForge`.
4. У застосунку відкрийте проєкт із директорії:

       fem_project/sierpinski_4

   або, якщо директорія у поточній версії має попереднє написання:

       fem_project/sierpisnki_4

5. Використовуйте цей проєкт для перегляду області, триангуляції, граничних умов та чисельних результатів, пов’язаних із експериментами бакалаврської роботи.
